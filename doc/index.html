connect (host, port, timeout, defer_hook)
Connect to a memcached server, returning a Memcached handle.

Parameters:

  • host Defaults to localhost.
  • port Defaults to 11211.
  • timeout Connection timeout. Defaults to 20 seconds
  • defer_hook An optional function, called to defer, enabling non-blocking operation.

Memcached:set (key, data, exptime, flags, noreply)
Set a key to a value.

Parameters:

  • key A key, which cannot have whitespace or control characters and must be less than 250 chars long.
  • data Value to associate with the key. Must be under 1 megabyte.
  • exptime Optional expiration time, in seconds.
  • flags Optional 16-bit int to associate with the key, for bit flags.
  • noreply Do not expect a reply, just set it.

Memcached:get (keys)
Get value and flags for one or more keys.

Parameters:

  • keys Key or {"list", "of", "keys"}.

Returns:

    For one key, returns (value, flags). For a list of keys, returns a { key1={data="data", flags=f }, key2=...} table.
